# Shoter Game

## Game Instructions

Welcome to the Shoter Game! In this game, you will navigate your character through an exciting landscape while aiming to shoot at targets. Follow the instructions below to get started.

## Controls
- **Arrow Keys**: Move your character around the screen.
- **Space Bar**: Shoot at the targets.

## Adding Custom Images
To add custom images to the game:
1. Place your images in the `assets` folder located in the root directory of the game.
2. Reference the images in your code using the corresponding file path, for example: `assets/image_name.png`.

Happy Gaming!
